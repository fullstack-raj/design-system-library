# Getting Started with Global Design System with Resuable Components
main gola is to create a simple resuable components

## Project Creation
```
npx create-react-app ui-global-system --template typescript
```

### Storybook setup
```
npx sb init
```
## Run Storybook
```
npm run storybook
```
